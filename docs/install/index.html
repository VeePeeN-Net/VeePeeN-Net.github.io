Linux 分图形化和没有图形化界面，所以比较麻烦，而且可能需要开启 tun 才能发挥最佳效果。

我们提供最佳实践：

第一步购买套餐

首先确保你手机安装了 TZ-VPN 的 APP。然后在手机的 APP 下单购买套餐；购买完成后，你的手机和 Linux 都可以同时使用，我们均不限制设备使用数。

第二步安装第三方客户端

  • Linux（有没有图形化均可），建议使用 UIF 。

  • OpenWrt(软路由)，建议使用 OpenClash 。

第三步将订阅导入到第三方客户端里

在手机 APP 里面选择你已购买的订阅，点击 分享，APP 会自动把订阅复制到手机的系统剪切板，粘贴到 Linux 里的第三方客户端即可。

第二步，使用​

点击 APP 内的 购买 按钮，选择适合的套餐，即可使用；
